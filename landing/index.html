Desktop app that indexes and searches your PDF documents. Fast full-text search. Your data stays on your computer.

The Problem

  • You have PDFs scattered across folders
  • Finding specific information takes too long
  • Your OS search is slow or doesn't work well
  • Cloud services require uploading sensitive documents

The Solution

  • Index all PDFs in any folder structure
  • Find content instantly with full-text search
  • Fast search built specifically for PDFs
  • Everything stays private on your machine

What It Does

Full-Text Search

Search the actual content inside your PDFs, not just filenames. Find any word or phrase instantly.

Private and Offline

No internet connection required. No cloud uploads. Your documents never leave your computer.

Fast Performance

Built with Rust for speed. Handles large document collections without slowing down your system.

Easy to Use

Select folders, search, done. No configuration needed. Works like you expect.

Advanced Search

Use AND, OR, NOT operators. Filter by date and file size. Sort results your way.

Automatic Indexing

Point to your folders once. The app indexes all PDFs, including subfolders.

How It Works

1

Select Folders

Choose folders containing your PDFs. The app will scan them including all subfolders.

2

Automatic Indexing

The app extracts text from each PDF and builds a search index. This happens once per folder.

3

Search Instantly

Type what you're looking for. Results appear as you type. Click any result to open the PDF.

Technical Information

System Requirements

  • Windows 10+, macOS 10.15+, or Linux
  • 50 MB disk space for the app
  • Additional space for the search index

Built With

  • Rust for performance and reliability
  • SQLite FTS5 for search capabilities
  • Tauri for native desktop integration
